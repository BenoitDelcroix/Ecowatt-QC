# Ecowatt-QC

Welcome to the documentation for the Ecowatt-QC project!

Ecowatt-QC is a Streamlit web application for visualizing and forecasting electricity demand in Quebec, inspired by the French Écowatt project.

- **Live grid status**
- **24-hour demand forecasts**
- **Open data sources**

## Quickstart

- Clone the repository
- Create a Python environment and install dependencies
- Run the Streamlit app

See the [Usage](usage.md) section for details.
